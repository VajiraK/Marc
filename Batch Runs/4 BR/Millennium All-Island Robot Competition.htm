MARC (Millennium All-Island Robot Competition) is a software competition for Sri Lankans. The challenge is to write a software robot using C, C++, VB or Java that can safely pickup some valuable packages from a wrecked spaceship. There are two categories in the competition enabling both school level competitors and open competitors to show their coding skills.

Follow these five easy steps and start coding the robot.
 
Get Registered
Register yourself in the competition. We will send you updates and news as the competition progress. Registered members have full access to the forum.
   
Download the SDK
Download the SDK and start coding your robot1. Make sure you go through the documentation and understand the concepts well.
   
Code Your Robot
Use your favorite programming language to code your robot2. The Spaceship program will show you how your robot performs. Read the FAQ and use the forum if you get in to trouble.
   
Upload Your Robot
Upload your robot by logging in to this website. If you didn't get registered before, it’s now or never.
   
Evaluate Your Score
See where you stand in the scores. At every batch-run, we will post the scores in the MARC Scoreboard. If you don’t see your name in the Top-Ten scorers list, well... you need to improve your robot. Start from Step 3 again. Even if you are in the top-ten list there may be still room for improvement.
   
  1. You don’t need to be a registered member to download the MARC SDK.
  2. The programming language should support the MARC API. Make sure your robot works in all sample maps that comes bundled with the SDK. If you are interested in creating your own maps, use the ShipBuilder program.
